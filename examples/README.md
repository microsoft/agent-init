# Primer Examples

This folder includes quick examples to help you get started.

## CLI usage

- Analyze a repo: primer analyze /path/to/repo
- Generate instructions: primer instructions --repo /path/to/repo
- Run readiness: primer readiness /path/to/repo
- Run evals: primer eval primer.eval.json --repo /path/to/repo

## Sample eval config

See primer.eval.json for a starter eval config you can customize.
